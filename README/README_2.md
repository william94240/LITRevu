## Installation

Install project with powershell windows:

```bash
    git clone https://github.com/coyote95/projet_9_oc.git
    cd projet_9_oc
    python -m venv ENV
    source ENV/Scripts/activate
    python  manage.py migrate
    pip install -r requirements.txt
    python manage.py runserver
```
